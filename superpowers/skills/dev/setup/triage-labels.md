# Triage Labels

这些 skills 使用五个标准 triage 角色来沟通。本文件把这些角色映射到当前仓库的 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的 label | 当前 tracker 中的 label | 含义 |
| ---------------------------- | ----------------------- | ---- |
| `needs-triage`               | `needs-triage`          | 维护者（Maintainer）需要评估这个 issue。 |
| `needs-info`                 | `needs-info`            | 等待报告者提供更多信息。 |
| `ready-for-agent`            | `ready-for-agent`       | 规格完整，已准备好交给 AFK agent。 |
| `ready-for-human`            | `ready-for-human`       | 需要人类实现。 |
| `wontfix`                    | `wontfix`               | 不会处理。 |

当某个 skill 提到一个角色，例如 “apply the AFK-ready triage label”，使用上表中对应的 label 字符串。

把右侧列改成你实际使用的标签名。
